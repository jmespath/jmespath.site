========
JMESPath
========

JMESPath (JSON Matching Expression paths) is a query language for search JSON
documents.  It allows you to declaratively extract elements from a JSON
document.  XPath, for JSON.

JMESPath includes:

* A formalized ABNF grammar
* A complete suite of compliance tests to ensure parity amongst libraries
* Libraries in numerous languages
* A complete specification
